# PFC_MINIPROJECT_AANCHALDHANGAR_10010

## Description
This is a C program that performs basic operations on an array.  
It can calculate the sum, find the maximum and minimum values, and count even and odd numbers.

## How to Run
1. Compile the program: `gcc miiniproject.c -o miiniproject`  
2. Run it: `./miiniproject`  
3. Enter numbers as input when asked.

## Example
Input: `1 2 3 4 5`  
Output:  
- Sum: 15  
- Maximum: 5  
- Minimum: 1  
- Even numbers: 2  
- Odd numbers: 3
